Getting Started
Here you will learn to develop an easy, fast, and fun workflow to help you design and host your webpage from any internet-connected computer with no proprietary software all for free.

Topics covered:

  • How to Run an HTML File
  • Recommended Text Editor: Drive Notepad
  • How To Host Your Own Web Page Through Google Drive
  • Recommended Image Editor: PicMonkey
How to Run an HTML File
Step 1: To run an HTML file on your computer you must have a browser and a text editor. A basic text editor in Microsoft Windows is Notepad and on a Mac Computer is TextEdit. There are also many other text editors you can use too with advanced features.
Step 2: Save your html file name with no capital letters and no spaces. You may use a dash in place of a space. Example: your-file-name.html
Step 3: Open up the folder where your HTML file is saved and open it using any web browser such as Internet Explorer, Safari, Google Chrome, etc. The browser you select will automatically interpret your code and display it.


Recommended Text Editor: Drive Notepad
Drive Notepad is a Google Drive App that allows you to code web pages. Log into your Google Drive, click ‘create’, then ‘connect more apps’. Search for Drive Notepad and install.


How To Host Your Own Website Through Google Drive
This tutorial will show you how you can easily host static web content using Google Drive by putting your files into a public Google Drive folder.
  1. Log in to your Google Drive account
  2. Select "Create > Folder"
  3. Name the new folder whatever you like
  4. Select the checkbox next to the new folder
  5. Click the "Sharing settings" icon
  6. Click "Change"
  7. Make your folder "Public on the web"
  8. Open the folder
  9. Find or create an html file and save it into this folder. Note:
    1. Name your home page 'index.html'
    2. Do not let Google Drive convert your HTML files into native Google Docs files. Adjust your settings appropriately.
  10. Your new folder now has a homepage. Now all you need is your url address. Click on the check box next to your html file. There should be a Details/Activity pane on the right. If not, click the 'i' button which looks like a lowercase 'i' with a circle around it. That is the 'Details and activity' icon.
  11. Finally click on the details tab, scroll down, and you will see your url. Click and you will see your web page come to life!


Now you have your own web site hosted through Google Drive!

Note: To make your website link more user-friendly you can use tinyurl.com

Recommended Image Editor: PicMonkey
PicMonkey is a great app to edit your photos. Features include crop, canvas color, rotate, exposure, colors, sharpen, and resize. It has an easy to use interface that I personally enjoy. There is an ad banner that does go along the bottom of the screen with an optional upgrade if you want to get rid of it. Other than that it's a great app. Log into your Google Drive, click ‘create’, then ‘connect more apps’. Search for PicMonkey and install.
