Color
  • CSS is used to set a color either for the foreground of an element (i.e., its text) or for the background of an element
  • CSS color is either a keyword or a numerical specification
  • When coding CSS, you will often need to define a color for an html element
  • In CSS, we define a color using either the hexadecimal color code, the RGB color code, or the actual color name
  • A benefit of using hexadecimal and RGB color codes is that we have many more options than just supplying a color name
  • CSS colors are defined using a hexadecimal (hex) notation for the combination of Red, Green, and Blue color values (RGB)
  • The lowest value that can be given to one of the light sources is 0 (hex 00). The highest value is 255 (hex FF)
  • We can specify your color values in various formats
  • The following table tells you all possible formats:
    Format/Syntax Example
    RGB %
    rgb(rrr%,ggg%,bbb%)    		 p {color:rgb(50%,50%,50%);}
    RGB
    Absolute rgb(rrr,ggg,bbb)    		 p{color:rgb(0,0,255);}
    Hex Code
    #RRGGBB    		 p{color:#FF0000;}
    Short Hex Code #RGB p{color:#6A7}
    Keyword
    aqua, black, etc    		 p{color:teal;}

Background
  • To define the background of an element we use the CSS background property
  • The background property is used to specify a number of other background properties
  • CSS properties used for background effects:
    • background-color
    • background-image
    • background-repeat
    • background-attachment
    • background-position
Background Color
  • To set the background-color of an element we use background-color property
  • The background-color property specifically declares a color for the background of an element
  • Syntax:
    • background-color: <color> | transparent | inherit
      • <color> - defines the color of the background
      • transparent - takes the same color as the element below it. It is the initial value of background-color
      • inherit - the element should have the same background-color setting as the parent. This property is not inherited
      • Example:
        <html>
        <head><style>
        body
        {
        background-color:#b0c4de;
        }
        </style></head>
        <body>
        <h1>CSS background-color Example</h1>
        <p>This is a sentence.</p>
        </body>
        </html>
Adding Background Color

Within a tag, such as div, you can add the attribute style and type in a color using the name or the hex code for it. That will give that section of your webpage a background of your choosing.
Example: <div style="background:yellow"><p>Text to have a background color</p></div>
Output:

Text to have a background color


Notice how the background goes all the way through the page horizontally. If you want the background color to cover a shorter width (or even height) use css to define exactly what you want to have the background color.
Example: <div style="background:yellow;width:100px;height:100px"><p>Text to have a background color</p></div>
Output:

Text to have a background color


Background Image
  • To set the background image of an element we use background-image property
  • The background-image property is used to add the background image of an element
  • background-image is the same as background-color, but it uses an image you specify instead of a color
  • CSS is used to set a background image for both the page and single elements on the page
  • background-image applies to all elements
  • Values for background image property are either ‘url’, to specify the image or ‘none’, when no image is used
  • Syntax:
    • background-image: <url> | none | inherit
    • <url> - the path to the image used as the background
    • none - no background image should display. It is the initial value of background image
    • inherit - the element should have the same background-image setting as the parent. This property is not inherited
    • Example:
      <html>
      <head><style>
      body
      {
      background-image:url(‘picture.png’);
      }
      </style></head>
      <body>
      <h1>Background Image Example</h1>
      </body>
      </html>
Background Repeat
  • To control the repetition of an image in the background we use background-repeat property
  • The background-repeat property is used to define the background image of an element
  • The background-repeat applies to all elements
  • Syntax:
    • background-repeat: repeat | repeat-x | repeat-y | no-repeat | inherit
    • repeat
      • used to display the background image horizontally and vertically
      • It is the initial value of background image
      • Example:
        <html>
        <head><style>
        body
        {
        background-image:url(‘picture.png’);
        background-repeat:repeat;
        }
        </style></head>
        <body>
        <h1>Background Image Repeat Example</h1>
        </body>
        </html>
    • repeat-x
      • It is used to display the background image horizontally
      • Example:
        <html>
        <head><style>
        body
        {
        background-image:url(‘picture’);
        background-repeat:repeat-x;
        }
        </style></head>
        <body>
        <h1>Background Image Repeat Example in x-axis</h1>
        </body>
        </html>
    • repeat-y
      • It is used to display the background image vertically
      • Example:
        <html>
        <head><style>
        body
        {
        background-image:url(‘picture’);
        background-repeat:repeat-y;
        }
        </style></head>
        <body>
        <h1>Background Image Repeat Example in y-axis</h1>
        </body>
        </html>
    • no-repeat
      • It is used to display the background image only once, no repetition of an image
      • Example:
        <html>
        <head><style>
        body
        {
        background-image:url(‘picture’);
        background-repeat:no-repeat;
        }
        </style></head>
        <body>
        <h1>Background Image Repeat Example in x-axis</h1>
        </body>
        </html>
    • Inherit
      • The element should have the same background-repeat setting as the parent
      • This property is not inherited
