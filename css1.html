Introduction to CSS
  • CSS stands for Cascading Style Sheets
  • Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation semantics of a document written in a markup language
  • CSS was created by Hakon Wium Lie and Bert Bos and was adopted as a World Wide Web Consortium (W3C) recommendation in late 1996
  • The CSS specifications are maintained by the W3C
  • Style sheets are simply text files (.css), composed of lines of code that tell browsers how to display an html page
  • CSS is a language that can be used to define styles against any html element
  • CSS is used to add layout to your websites
  • CSS is a language that is used to define layout of html documents. For example, CSS covers fonts, colors, margins, lines, height, width, background images, advanced positions, and many other things
  • The main purpose of CSS is to define the presentation of html elements
  • CSS are a series of instructions that specify how markup elements should appear on a web page
Types of CSS
  • There are 3 types of CSS
    • Inline
    • Internal (Embedded)
    • External
  • Inline Style
    • Inline style is defined directly in the tag on document of a particular page element
    • Inline styles affect only the tag they are applied to
    • Example 1: <a href=”” style”text-decoration:none;”></a>
    • Example 2: <p style=”font-size:14px; color:green;”></p>
    • Advantage - Can be useful for unique changes to an elements styling within a document
    • Disadvantage - Hard to maintain consistency across related elements
  • Internal Style
    • Internal style is defined in the <head> section of a given document
    • Internal styles affect only the tags on the page they are embedded in
    • Should be used when a single document has a unique style
    • Defined in the head section by using the <style> tag
    • Example: <head>
      <title>Internal Style</title>
      <style type=”text/css”>
      p {margin-left: 20px}
      </style>
      </head>
    • Advantages - Manage styling within a single document
    • Disadvantages - Hard to maintain consistency across several related web pages
  • External Style
    • External style is defined on the external style sheet, which is linked to a web page
    • External style is known as linked style sheet
    • External styles are used to define in a separate document and then attached to various web documents
    • Each web page must link to the style sheet using the <link> tag
    • Browser reads styles definitions from the CSS file
    • Example: <head>
      <link rel=”stylesheet” type=”text/css” href=”mystyle.css” />
      </head>
    • Advantage - Can link to multiple documents
    • Disadvantage - Need to use consistent markup on multiple pages
History of CSS
  • Style sheets have existed since the beginnings of SGML in the 1970s
  • Nine different style sheet languages were proposed to the W3C
  • CSS level 1 - 1996; CSS level 2 - 1997
Use of CSS
  • The main use for CSS is to improve html web pages by decorating them, and displaying them in a better way than they would normally do
  • A CSS is a single page of formatting instructions that can control the appearance of many html pages at once
  • CSS is used to define how web pages should look when viewed in other media than a web browser
The Benefits of CSS
  • CSS saves you time
  • CSS saves your visitors time
  • Pages load faster
  • Pages and sites are easier to maintain
  • Easy maintenance
  • Superior styles to html
  • Multiple device compatibility
  • Global web standards
  • CSS gives you design flexibility
  • CSS also allows websites to be planned, produced, and maintained in much less time
  • You can change the appearance of your entire document by adjusting a few rules in a style sheet
  • Separation of html content from appearance
Disadvantages of CSS
  • Browser compatibility
  • Number of multiple backgrounds per element
  • Lack of variables
  • Collapsing margins
  • Vertical control limitation
How Does CSS Work?
  • When the web browser displays a document, it must combine the document's content with its style information
  • It processes the document in two stages:
    • Thebrowser converts the markup language and the CSS into a structure called the DOM (Document Object Model)
    • The DOM represents the document in the computer's memory
    • It combines the document's content with its style
    • The browser displays the contents of the DOM
  • Example: Instead of defining fonts and colors each time you start a new table cell, you can define a style and then, simply refer to that style in your table cells
    • With HTML:
      <html>

CSS Syntax
  • The CSS syntax consists of a set of rules. These rules have 3 parts:
    • Selector
    • Property
    • Value
  • Syntax: selector {property: value;}
  • Example: BODY {color:yellow;}
  • Selector:
    • The selector is normally the html element/tag you wish to define
    • Selector is usually standard html elements (h1, p, body, table, etc.)
    • We change the style of selectors by changing one of many attributes
  • Property:
    • The property is the attribute you wish to change
    • Every property has the value
    • If the value is multiple words, put quotes around the value
    • Properties are names of attributes that are used to describe an object (color, text, font-face, font-size, etc.)
    • Property is any characteristic of the selector that we wish to change like color, size, etc.
  • Value:
    • Values are “describing” properties (yellow, bold, fixed, arial, etc.)
    • Value is what you wish the style to be
Comment in CSS
  • Comment is used to explain our code and may help us when we edit the source code at a later date
  • Comments are ignored by browsers
  • Everything that lies between these delimiters is the comment and should be ignored by the CSS parser
  • Comments can span multiple lines, but may not be nested
  • Comments can include CSS elements that we don’t want in the document but don’t want to delete completely
  • Comments are used to write complicated CSS to add clarification
  • Comments can also include meta information like:
    • Author
    • Date created
    • Copyright information
  • Syntax: // single-line comment
  • /* This is a comment that spans multiple lines */
How to Add Comments in a CSS Document
  1. Add the first part of the CSS comment /*
  2. Write the text or information you want to be hidden
  3. Close the comment */

How to Move Change the Position of objects on a Webpage

You can move the position of any object you have on your webpage by using CSS. By using a style attribute you can define exactly where you want your objects to be. We will use the following code as an example: <div style="position:absolute;top:100px;left:200px">

Text to be moved from CSS code


The div tag has a style attribute which allows you to add css code. The 'position:absolue' means relative to the top left of the page the object is being moved from the top 100 pixels (meaning it's moving down 100 pixels) and from the left 200 pixels (meaning it is moving to the right 200 pixels).

Common CSS Properties and Values
  • Position
    • Example: position:absolute
      • This means that the selector will not move regardless of how the window is sized
    • Top
      • Example: top:100px
        • This means the selector (h1, p, body, table, etc.) is positioned 100 pixels from the top of the page
    • Left
      • Example: left:50px
        • This means the selector (h1, p, body, table, etc.) is positioned 50 pixels from the left of the page
    • Example: <h1 style="position:absolute; top:100px; left:50px">
  • Background
    • Example: background:yellow
    • Width
      • Example: width:400px
        • This means that the yellow background only stretches 400 pixels wide
    • Height
      • Example: height:300px
        • This means that the yellow background only stretches 300 pixels wide
    • Example: <div style="background:yellow; width:400px; height:300px">
  • Border - solid, dotted
    • Example: border: 1px solid orange
      • This means that the border will be 1 pixel in width and will be a solid orange color
    • Example: <div style="border: 1px solid orange;">
  • Padding
    • Example: padding:15px
      • This means that there will be a space of 15 px between the inside of the border and the text within the border
  • Margin
    • Example: margin:20px
      • This means that the border will be placed 20 pixels away from the left of the page
    • There are also margin properties that give you margin only in one direction such as:
      • margin-top
      • margin-left
      • margin-right
  • Example: <div style="border: 1px solid orange; padding:15px; margin:40px">
  • Minimum Height
    • Example: min-height:350px
  • Maximum Height
    • Example: max-height:75px
  • Style Tag - <style></style>
    • Style tags allows you to define a class. You can use this class to set the style in another part of your code such as a <div>.
    • Example of a class:
      <head>
      <title>This is a title</title
      <style>
      .newjersey{
      color:blue
      }
      </style>
      </head>
      <body>
      <div class=newjersey>
      <h1>Testing the Style tag</h1>
      <p>This is a paragraph of text.</p>
      </div>
      • <li>Example of an id:
      <head>
      <title>This is a title</title
      <style>
      #newjersey{
      color:orange
      }
      </style>
      </head>
      <body>
      <div id=newjersey>
      <h1>Testing the Style tag</h1>
      <p>This is a paragraph of text.</p> </div>
  • Elements can also be changed with the <style> tag
    • If you want all <div>’s to have a border you can set that within the <style> tag
    • Example:
      <head>
      <title>This is a title</title
      <style>
      #newjersey{
      color:orange
      }
      div{
      border:1px dotted green;
      </style>
      </head>
      <body>
      <div id=newjersey>
      <h1>Testing the Style tag</h1>
      <p>This is a paragraph of text.</p>
      </div>
