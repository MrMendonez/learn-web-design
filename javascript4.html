jQuery Tutorial for Beginners



The DRY Principle - Don't Repeat Yourself The DRY Priciple, which stands for Don't Repeat Yourself, is a principle used in computer science. Essentially, you do not want to have code that repeats itself. For instance, when making a web page the code for your navigation bar may repeat itself on every single web page you have for your site. This could lead to problems if you want to make a change to your navigation bar. If you want to add a new menu button on your navigation bar then you would have to go to every html file you have to add that line or lines of code. This can be very cumbersome if you have many web pages to update. By using jQuery - a JavaScript library - you can code your navigation bar once on a seperate html file and call it using a div id. That way when you make a change to your navbar you only have to change it once and it will be applied to all your web pages. It is the same idea as using an external CSS file.
